# 📝 Java File I/O – Notes App

## 🎯 Objective
A simple text-based Notes Manager built using Java File I/O concepts.

## ⚙️ Features
- Add new notes
- View all saved notes
- Notes are stored in `notes.txt`

## 💻 Technologies Used
- Java
- FileWriter / BufferedReader
- Exception Handling

## 🧠 Key Concepts
- File I/O
- Try-with-resources
- Append vs Overwrite
- IOException handling
